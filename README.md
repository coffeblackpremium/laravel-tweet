- Simple demo app for a tweet (Similar to twitter tweet). 


### How to execute

pre-requirements:
- PHP >= 8
- nodejs > 18.0

1) Clone repository
```git
git clone: https://github.com/coffeblackpremium/laravel-tweet.git && cd laravel-tweet/
```

2) install Dependencies
```bash
composer install && npm install
```
3) Run
```bash
php artisan serve
```
